# SALT PM

이 폴더는 SALT의 기능 기획서와 제품 기능 인벤토리를 관리한다.

먼저 볼 문서:

- `AGENTS.md` — PM 문서 작업 하네스
- `.codex/rules/feature-planning.md` — 기능 기획서 작성 규칙
- `.codex/rules/feature-inventory.md` — 전체 기능 파악 규칙
- `.codex/rules/traceability.md` — 화면/API/DB 추적 규칙
- `features/current-feature-map.md` — 현재 기능 맵
- `templates/feature-spec-template.md` — 기능 기획서 템플릿

기능 변경이 생기면 `features/current-feature-map.md`와 관련 `FEATURE-*` 문서를 함께 업데이트한다.
