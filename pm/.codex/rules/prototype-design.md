# PM 프로토타입/디자인 규칙

## 목적

PM 프로토타입은 실제 구현 앱이 아니라 기능 기획을 빠르게 검증하기 위한 화면이다. 단, 백엔드 응답과 WebSocket 메시지는 실제 계약과 같은 shape로 맞춘다.

## 위치

```text
pm/prototype/
  index.html
  src/
    app.js
    styles.css
    dummy/
```

## 원칙

- 실제 서버를 호출하지 않는다. 모든 데이터는 더미 계약으로 제공한다.
- 더미 REST 응답은 `success`, `message`, `data` envelope을 따른다.
- 더미 WebSocket 이벤트는 실제 BFF 메시지 타입을 따른다.
- 화면은 기능 명세를 검증하기 위한 수준으로 만들되, 축약판으로 끝내지 않는다.
- 백엔드에 이미 존재하는 투자 기능은 프로토타입에 모두 등장해야 한다.
- 목록 기능은 반드시 상세보기 화면 또는 모달 진입점을 포함한다.
- 생성/수정이 필요한 기능은 실제 저장을 하지 않더라도 입력 모달을 포함한다.
- 알림, 플레이북, 뉴스, 북마크, 드로잉처럼 상태가 있는 기능은 `ON/OFF`, `active/draft`, `bookmarked` 같은 상태 UI를 포함한다.
- 구현 앱의 디렉터리나 실제 서버 코드를 수정하지 않는다.
- PM 프로토타입에서 결정한 화면/정책은 기능 기획서에 반영한다.

## 더미 REST 계약

```js
{
  success: true,
  message: "success",
  data: {}
}
```

목록 응답은 실제 백엔드와 같은 pagination 구조를 우선한다.

```js
{
  success: true,
  message: "success",
  data: {
    items: [],
    pagination: {
      page: 1,
      limit: 20,
      total: 0,
      totalPages: 0
    }
  }
}
```

## 더미 WebSocket 계약

가격 업데이트:

```js
{
  type: "price_update",
  data: {
    symbol: "BTC",
    currentPrice: 134200000,
    change24h: 2.4,
    timestamp: "2026-05-24T00:00:00.000Z"
  }
}
```

캔들 업데이트:

```js
{
  type: "candle",
  symbol: "BTC",
  timeframe: "5m",
  data: {
    timestamp: "2026-05-24T00:00:00.000Z",
    open: 133000000,
    high: 134500000,
    low: 132900000,
    close: 134200000,
    volume: 12.4
  }
}
```

## 디자인 규칙

- PM이 기능을 설명하기 쉬운 dashboard 형태를 우선하되, 기존 `salt-microFe` investment 화면의 톤을 기준으로 한다.
- 첫 화면에서 핵심 KPI를 보여주고, 탭으로 시장/상세, 포트폴리오, 인텔리전스, 알림/플레이북, 피드/뉴스/드로잉, 계약 확인을 모두 이동할 수 있어야 한다.
- 색상은 기존 investment 화면처럼 `#F2F4F6` 배경, 흰 카드, `#007AFF` 포인트, 상승 빨강, 하락 파랑을 기본으로 한다.
- 데이터가 없는 상태, 로딩, 위험 상태, 알림 상태를 반드시 표현한다.
- 복잡한 기능은 카드와 표를 조합하고, 과도한 장식은 피한다.
- 프로토타입 화면에 “이 기능이 사용자에게 주는 이점”을 짧게 표시한다.

## 완료 조건

- 프로토타입이 브라우저에서 실행된다.
- 시장 조회, 관심종목, 실시간 가격/차트, 자산 상세, 포트폴리오, 보유 기록, 시장 인텔리전스, 투자 인사이트, AI 코치, 알림, 플레이북, 피드, 뉴스/북마크, 차트 드로잉, 기술 지표, REST 계약, WebSocket 계약이 화면에 보인다.
- 각 목록 row는 상세보기 또는 상태 변경 모달을 열 수 있어야 한다.
- 더미 API 응답과 더미 socket 이벤트를 화면에서 확인할 수 있다.
- PM 보고서 템플릿과 기능 명세가 함께 업데이트된다.
